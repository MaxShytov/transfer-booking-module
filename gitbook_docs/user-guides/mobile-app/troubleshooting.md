# Troubleshooting

Common issues and solutions for the 8Move Transfer mobile app.

## Quick Diagnostics

Before diving into specific issues, try these general fixes:

### Universal Solutions

**The "Turn It Off and On Again" Approach:**

1. **Force Close App**
   - iOS: Swipe up from bottom, swipe app up
   - Android: Recent apps button, swipe app away
   - Reopen app

2. **Check Internet Connection**
   - Verify WiFi or mobile data is on
   - Try switching between networks
   - Test with browser (open any website)

3. **Update App**
   - Check App Store/Play Store
   - Install latest version
   - Many bugs fixed in updates

4. **Restart Device**
   - Power off completely
   - Wait 10 seconds
   - Power back on

5. **Clear Cache** (Android)
   - Settings → Apps → 8Move Transfer
   - Storage → Clear Cache
   - Don't clear data (loses login)

## Authentication Issues

### Can't Login

**"Invalid credentials" error**

**Possible Causes:**
- Wrong email or password
- Caps Lock enabled
- Copy-paste added spaces
- Account doesn't exist

**Solutions:**
1. ✅ Double-check email spelling
2. ✅ Verify password (case-sensitive)
3. ✅ Turn off Caps Lock
4. ✅ Type manually (don't paste)
5. ✅ Try "Forgot Password" (coming soon)
6. ✅ Verify you registered this email

---

**"Network error" during login**

**Solutions:**
1. Check internet connection
2. Try different network (WiFi ↔ Mobile data)
3. Disable VPN if using one
4. Check if server is accessible
5. Wait a few minutes and retry

---

### Can't Register

**"Email already exists"**

**This means:**
- Account with this email already registered
- You may have forgotten you registered

**Solutions:**
1. ✅ Try logging in instead
2. ✅ Use different email
3. ✅ Contact support to verify account

---

**"Invalid email format"**

**Solutions:**
- Use proper format: name@domain.com
- No spaces before/after
- Valid domain extension required
- Can't use special characters except @ and .

---

**"Passwords don't match"**

**Solutions:**
- Retype both password fields
- Check Caps Lock
- Copy first password field, paste in confirm
- Make sure no trailing spaces

---

### Logged Out Unexpectedly

**Session expired**

**Why it happens:**
- Security measure after inactivity
- App updated
- Device restarted

**Solution:**
- Simply log in again
- Your bookings are safe

---

## Booking Issues

### Can't Create Booking

**Wizard won't proceed to next step**

**Common reasons:**

**Step 1 - Route Selection:**
- ❌ Pickup not selected
- ❌ Dropoff not selected
- ❌ Same pickup and dropoff
- ❌ Date/time not set
- ❌ Passengers = 0
- ✅ Ensure all required fields filled

**Step 2 - Vehicle Selection:**
- ❌ No vehicle selected
- ❌ All vehicles disabled
- ✅ Select an enabled vehicle

**Step 5 - Review:**
- ❌ Terms not accepted
- ❌ Privacy policy not checked
- ✅ Check both boxes

---

**"No vehicles available" message**

**This happens when:**
- Too many passengers for all vehicles
- Too much luggage for all vehicles
- Combination exceeds largest vehicle

**Solutions:**
1. Reduce passenger count
2. Reduce luggage count
3. Split into two bookings
4. Contact operator for special arrangements

---

**Price keeps changing**

**This is normal!** Price updates in real-time when you:
- Change vehicle class
- Adjust passenger count
- Select different dates (seasonal)
- Change pickup time (time multipliers)
- Add/remove extras

**Expected behavior:**
- Dynamic pricing reflects all factors
- Final price locked at confirmation
- Review complete breakdown before confirming

---

### Booking Not Appearing

**Just created booking but not in "My Bookings"**

**Solutions:**
1. Pull down to refresh list
2. Check you confirmed (not just reviewed)
3. Verify success message appeared
4. Check you're logged in
5. Look for confirmation email
6. Contact support with booking reference

---

### Can't View Booking Details

**Tap booking card but nothing happens**

**Solutions:**
1. Check internet connection
2. Pull to refresh list first
3. Try different booking
4. Force close and reopen app
5. Clear cache (Android)

## Map & Location Issues

### Map Not Loading

**Blank map in booking wizard**

**Solutions:**
1. ✅ Check internet connection (maps require data)
2. ✅ Allow location permissions if prompted
3. ✅ Wait longer (maps take time on slow connections)
4. ✅ Scroll down and back up to trigger reload
5. ✅ Skip map (not required for booking)

---

**Route not showing**

**Solutions:**
- Ensure both pickup and dropoff selected
- Wait a few seconds for route calculation
- Zoom out to see full route
- Map might be loading (give it time)

---

### GPS Location Issues

**"Current location" not working** *(Coming soon feature)*

**Solutions:**
1. Enable location services:
   - iOS: Settings → Privacy → Location Services → 8Move
   - Android: Settings → Location → App permissions → 8Move
2. Grant "While Using App" permission
3. Go outside if indoors
4. Wait for GPS signal
5. Restart device if persistent

3. Check device storage (need ~500MB free)
4. Close other apps
5. Update to latest version
6. Clear cache (Android)
7. Reinstall app (last resort)

**Solutions:**
1. ✅ Update to latest version
2. ✅ Restart device
3. ✅ Free up storage space
4. ✅ Check for OS updates
5. ✅ Reinstall app

## Network & Connection

### "No internet connection" Error

**Even though you have internet**

**Solutions:**
1. ✅ Toggle airplane mode on/off
2. ✅ Switch WiFi ↔ Mobile data
3. ✅ Forget and rejoin WiFi network
4. ✅ Reset network settings
5. ✅ Check if other apps work
6. ✅ Disable VPN temporarily
7. ✅ Contact your ISP

---

### Slow Loading

**Data taking forever to load**

**Solutions:**
- Check internet speed (use speedtest)
- Move to area with better signal
- Switch to WiFi if on mobile data
- Close bandwidth-heavy apps
- Try later (server might be busy)

## Error Messages

### Common Error Messages Explained

**"Something went wrong"**
- Generic error
- Usually temporary
- Try again in a few minutes
- Check internet connection

**"Invalid data"**
- Form fields incorrect
- Check required fields
- Verify formats (email, phone)
- Look for red error indicators

**"Session expired"**
- Logged out due to inactivity
- Security measure
- Simply log in again

**"Service unavailable"**
- Server temporarily down
- Maintenance in progress
- Try again in 10-15 minutes

**"Rate limit exceeded"**
- Too many requests
- Wait 60 seconds
- Try again

## Still Having Issues?

### Before Contacting Support

Gather this information:
1. **Device info:**
   - Device model (e.g., iPhone 12, Samsung Galaxy S21)
   - Operating system version (iOS 15, Android 12)
   - App version (found in More → About)

2. **Issue details:**
   - What you were trying to do
   - Exact error message
   - When it started happening
   - Steps to reproduce

3. **Booking info** (if applicable):
   - Booking reference number
   - Date and time of booking
   - Email used

### Contact Support

**In-App:**
1. Open app
2. Go to More tab
3. Tap Help & Support
4. Submit issue (coming soon)

**Email:**
- support@8move-transfer.com
- Include all gathered information
- Attach screenshots if possible

**Response Time:**
- Usually within 24 hours
- Urgent: Mark as high priority
- Business hours: Faster response
