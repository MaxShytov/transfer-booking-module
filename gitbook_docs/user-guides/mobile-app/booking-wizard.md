# Booking Wizard

The booking wizard guides you through creating a transfer booking in 5 easy steps. Your progress is shown at the top of each screen.

<figure><img src="../../.gitbook/assets/Mobile App. Booking Wizard. 5 steps.png" alt="" width="375"><figcaption><p>Progress indicator showing 5 steps of the booking process</p></figcaption></figure>

## Overview

The wizard ensures you provide all necessary information while calculating your price in real-time. You can go back to previous steps to make changes at any time.

### The 5 Steps

1. **Route Selection** - Configure pickup, dropoff, dates, passengers, and luggage
2. **Vehicle Selection** - Choose your preferred vehicle class
3. **Extras Selection** - Add optional services
4. **Passenger Details** - Provide contact information
5. **Review & Confirm** - Final review and booking confirmation

### Progress Indicator

At the top of each step:

* Visual progress bar showing current step
* Step number and total (e.g., "Step 2 of 5")
* All steps labeled for easy reference

### Price Breakdown

**Complete transparency before booking:**

#### For Regular Customers

Simplified pricing display:

* Base price
* Selected extras
* Round trip discount (if applicable)
* **Final Total**

#### For Admin/Manager Users

Detailed breakdown showing all multipliers:

| Component                | Description                   | Value |
| ------------------------ | ----------------------------- | ----- |
| **Pricing Type**         | Fixed Route or Distance-based | -     |
| **Route/Distance**       | Route name or km              | -     |
| **Base Price**           | Starting price                | CHF X |
| **Vehicle Multiplier**   | Selected vehicle class        | √ó 1.5 |
| **Passenger Multiplier** | Capacity utilization          | √ó 1.1 |
| **Seasonal Multiplier**  | Peak/off-peak                 | √ó 1.2 |
| **Time Multiplier**      | Early/late/rush hour          | √ó 1.1 |
| **Subtotal**             | Base √ó all multipliers        | CHF Y |
| **Extra Services**       | Itemized list                 | CHF Z |
| **Extras Total**         | Sum of extras                 | CHF Z |
| **Round Trip Discount**  | If applicable                 | - 10% |
| **Final Total**          | Complete price                | CHF W |

**Formula:**

```
Subtotal = Base √ó Vehicle √ó Passengers √ó Season √ó Time
Final = (Subtotal + Extras) √ó RoundTripDiscount
```

### Confirmation Process

<figure><img src="../../.gitbook/assets/Confirmation booking.png" alt="" width="375"><figcaption></figcaption></figure>

**Final Steps:**

1. **Review all details carefully**
   * Check dates, times, locations
   * Verify contact information
   * Confirm price is acceptable
2. **Accept agreements**
   * Check both required boxes
   * Ensure you understand terms
3. **Tap "Confirm Booking"**
   * Button becomes enabled
   * Shows loading indicator
   * Processes your booking
4. **Wait for confirmation**
   * Usually takes 1-3 seconds
   * Don't close app during processing
5. **Success Dialog**
   * Shows your booking reference number
   * Note it down for future reference
   * Can also be found in My Bookings
6. **Automatic Navigation**
   * Redirects to My Bookings tab
   * Your new booking appears at top
   * Status: "Pending" (awaiting operator confirmation)

## Common Questions

**Can I book for someone else?** Yes, enter their contact details in Step 4.

**What if my flight is delayed?** Provide flight number - system tracks automatically and adjusts pickup time.

**Can I add stops?** Add in "Special Requests" - operator will confirm and may adjust price.

**How do I cancel?** Contact operator - cancellation policy applies.

**When will I be charged?** Payment methods and timing depend on operator configuration. Currently payment is handled offline.
