## Troubleshooting

### Can't Update Information

**If profile changes won't save:**
1. Check internet connection
2. Verify field formats (email, phone)
3. Try logging out and back in
4. Clear app cache
5. Contact support if persistent

### Language Not Changing

**Solutions:**
1. Force close app
2. Reopen app
3. Language should apply
4. Reinstall if needed

### Logout Issues

**If can't logout:**
1. Force quit app
2. Reopen
3. Try logout again
4. Reinstall if needed

## Account Deletion

### Request Account Deletion

To permanently delete your account:

1. Contact support via email or in-app
2. Provide account email
3. Confirm deletion request
4. Account deactivated within 7 days
5. Data deleted per retention policy

### What Gets Deleted

**Removed:**
- Personal information
- Login credentials
- Profile data
- Communication preferences

**Retained (Legal Requirements):**
- Completed booking records (7 years)
- Transaction history (accounting)
- Essential audit logs

### Before Deleting

**Consider:**
- Cannot be undone
- Lose booking history
- Need new account to rebook
- Outstanding bookings affected

**Alternatives:**
- Logout instead of deletion
- Change email/phone if privacy concern
- Deactivate notifications
- Contact support for specific concerns
