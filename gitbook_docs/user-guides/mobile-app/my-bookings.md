# My Bookings

View, manage, and track all your transfer bookings in one convenient location.

## Overview

The My Bookings section provides complete visibility into your booking history, from pending reservations to completed transfers. Access it from the second tab in the bottom navigation bar.

## Common Scenarios

### Booking Still Pending

**If pending for over 24 hours:**
1. Pull to refresh
2. Check email for updates
3. Contact operator if urgent
4. Have booking reference ready

### Need to Make Changes

**Modification process:**
1. Open booking details
2. Note booking reference
3. Contact operator support
4. Provide reference number
5. Discuss needed changes

**Common changes:**
- Date/time adjustments
- Passenger count
- Luggage requirements
- Special requests
- Flight number updates

### Cancellation

**How to cancel:**
1. Contact operator (contact in app or email)
2. Provide booking reference
3. Specify reason
4. Confirm cancellation
5. Ask about refund policy

**Policies:**
- Vary by operator
- May depend on timing
- Check terms & conditions
- Early cancellation usually better

## Troubleshooting

### Booking Not Appearing

**Possible causes & solutions:**

1. **Not logged in**
   - Log in to your account
   - Bookings tied to account

2. **Network issue**
   - Check internet connection
   - Pull to refresh
   - Try switching WiFi/mobile data

3. **Booking not confirmed**
   - Check email for confirmation
   - May be processing
   - Contact support if delayed

4. **Wrong account**
   - Verify logged in to correct account
   - Check email used for booking

### Status Not Updating

**Solutions:**
1. Pull down to refresh list
2. Close and reopen app
3. Check internet connection
4. Wait a few minutes and retry
5. Contact support if persistent

### Price Discrepancy

**If price differs from quote:**
1. Open booking details
2. Review price breakdown
3. Check for applied multipliers
4. Verify extras included
5. Contact operator for clarification
